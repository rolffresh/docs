---
title: "Open links in desktop app"
description: "Open Linear issues from LinCal directly in the Linear desktop app instead of the browser."
---

If this is enabled in Linear, clicking a Linear issue link in LinCal will open in the desktop app (for the same workspace) when possible.

![Filter menu screenshot](/images/using-lincal/LinCal-OpenLinksDesktop.png)

## Enable in Linear

1. Open linear.app in your browser.
2. Go to Settings.
3. Open General preferences.
4. Scroll to Desktop application.
5. Turn on Open in desktop app.

## What to expect
- When enabled, issue links from LinCal open in the Linear desktop app.
- If the desktop app is not installed or not available, links open in the browser.
- This behavior is controlled by Linear’s setting, not a separate LinCal toggle.

## Troubleshooting
- Make sure you are signed into the same workspace in both LinCal and the Linear desktop app.
- Confirm Open in desktop app is enabled in Linear settings.
- If links still open in browser, restart the Linear desktop app and try again.
