---
title: "Load Linear Views"
description: "Use your Linear views in LinCal."
---

LinCal can load issues from views you already use in Linear.

## What views appear

In the **Views** filter, you may see:

- **Personal views** — your private Linear views.
- **Workspace views** — views shared across the Linear workspace.
- **Team views** — views attached to a specific Linear team. Select that team first to see its team-specific views.

## Use a Linear view

1. Open **Filter...**.
2. Select **Views**.
3. Choose a view.

LinCal uses the view definition from Linear and places matching issues on the calendar by due date.

## How views behave

- Issues with due dates appear in the calendar.
- Issues without due dates appear in **Tasks without a due date** if they were updated recently.
- Selecting a view clears project, status, assignee, cycle, and label filters.
- Access follows Linear permissions. If you can see the view and its issues in Linear, LinCal can load them.
