description: "Use Recents and Favorites to reapply filter combinations in one click."
---

When you build a filter setup (for example Team + Cycle + Assignee), LinCal remembers it in Recents so you can quickly return to it later.
Star a combination to keep it pinned as a Favorite.

![Filter menu screenshot](/images/using-lincal/LinCal-recents.png)

## Reuse a recent filter combination
1. Open Filter.
2. Go to Recents.
3. Select a saved combination to apply it instantly.

## Star a combination
1. Apply the filters you want.
2. In the filter list, click the star icon on that combination.
3. It appears in your favorites so it is always easy to access.

## Notes
- Recents help you jump back into work without rebuilding filters.
- Favorites are best for views you use every day.
- If no filters are set, LinCal defaults to showing work from your own teams.
