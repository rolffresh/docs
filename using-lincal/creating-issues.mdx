description: "Create new Linear tasks directly from within LinCal."
---

## Create a task from a date

1. Hover a day cell and click **New Task**.
2. Enter the task title and press **Enter**.
3. Select a team (if needed).
4. Optionally set properties:
   - Assignee
   - Project
   - Labels
   - Cycle
5. Press **Create task** (or press **Enter**).

### Notes

- Tasks are created in Linear and appear in the selected date.
- If your filters already narrow to a single team, LinCal may preselect that team automatically.

## Edit a task (right-click)

1. Right-click a task card in the calendar.
2. The task properties panel opens in edit mode.
3. Update properties such as assignee, project, labels, or cycle.
4. Click **Save changes**.

### Notes

- This flow is for quick property updates from the calendar.
- Changes sync back to Linear.

## Change status from the task card

1. On a task card, click the **status icon** (left side of the card).
2. Select the new status from the status menu.
3. The card updates immediately.

### Notes

- Available statuses come from the task’s team workflow in Linear.
- This is the fastest way to move work between states from calendar view.
